== Installation ==

1. Upload the entire 'woocommerce-all-currencies' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. All currencies will be automatically added to WooCommerce -> General Options.

== Frequently Asked Questions ==

= Can I change symbol for currency? =

You need [WooCommerce All Currencies Pro plugin](http://www.algoritmika.com/shop/wordpress-woocommerce-all-currencies-pro-plugin) to change these settings.
